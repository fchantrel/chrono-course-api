# chrono-course-api
API python pour générer une liste de participants (aléatoire) à une course

## Installation
Installer les librairies nécessaires avec pip

## Demarrer l API
'''
python3 api.py
'''

## Tester l API
'''
curl -i -X GET http://localhost:9090/chrono-course-api-1/participants?course=rennes
'''
